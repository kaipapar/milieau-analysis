# Milieau analysis
## The plan is to:
1. Scrape property listings in Turku for their specs.
2. Create a dataset from those.
3. Combine the property dataset with the [Turk milieau dataset from avoindata.fi](https://www.avoindata.fi/data/en_GB/dataset/turun-keskustan-miljootyypitys).
4. Analyze whether or not the milieaus / environment types have any effect on property listings.
